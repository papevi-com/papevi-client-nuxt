# papevi-client-nuxt

Nuxt 4 application scaffolded with `npm create nuxt@latest`.

## Setup

```bash
npm install
cp .env.example .env
```

Set `API_TOKEN` in `.env` for requests to `https://api.papevi.com`.

## Development

```bash
npm run dev
```

## Quality checks

```bash
npm run lint
npm run format:check
```

## Production

```bash
npm run build
npm run preview
```
